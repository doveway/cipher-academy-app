## Image Guidelines

### Format
- Use PNG for graphics with transparency
- Use WebP for photographs and complex images
- Use SVG when possible for icons (place in `assets/svg/`)

### Size Optimization
- Compress all images before adding to project
- Use appropriate resolution for mobile devices
- Provide @2x and @3x versions for iOS if needed

### Dark Academia Aesthetic
- Muted color palette: burgundy, forest green, navy, brass
- Vintage/antique style elements
- High contrast for readability
- Sophisticated, not childish
